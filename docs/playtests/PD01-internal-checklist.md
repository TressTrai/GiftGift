# Внутренний плейтест: чеклист

## Механики

- [ ] Игрок может зарегистрироваться или войти за 30-60 секунд.
- [ ] Игрок понимает, что предметы на сцене можно собирать касанием.
- [ ] Собранный предмет появляется в инвентаре.
- [ ] Предмет можно подарить другому игроку.
- [ ] Подарок нельзя отправить самому себе.
- [ ] Нераскрытый подарок раскрывается за короткую анимацию и попадает в инвентарь.
- [ ] Подарок от другого игрока засчитывается в личную тройку, если совпадает с целью.
- [ ] Дарение увеличивает визуальный коллективный прогресс без показа точного числа.
- [ ] Передаренный подарок превращается в другой подарок из каталога.

## Стабильность

- [ ] Сбор одного и того же предмета двумя игроками не ломает состояние.
- [ ] После перезагрузки страницы сохраняются инвентарь, подарки и личная тройка.
- [ ] Основные экраны работают на мобильном экране в portrait.
- [ ] Кнопки и карточки не перекрываются и нажимаются пальцем.
- [ ] Нет блокирующих ошибок в авторизации, сборе, дарении и раскрытии.

## Готовность

- [ ] За один короткий заход игрок может: войти, собрать предмет, подарить, раскрыть подарок.
- [ ] В игре достаточно тестовых пользователей для проверки обмена.
- [ ] Каталог подарков загружается полностью.
- [ ] Можно вручную проверить 10-минутный респавн или его тестовый ускоренный режим.
- [ ] Сборка готова к передаче внешним участникам.
