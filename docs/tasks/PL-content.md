# Production List: Куратор подарочного контента

## Must

- Собрать список 60-70 кандидатов подарков.
- Отобрать финальные 50 подарков.
- Проверить, что подарки не делятся на мужские и женские.
- Убрать обидные, слишком личные и военные формулировки.
- Убрать подарки, понятные только одному человеку.
- Убрать слишком абстрактные подарки без узнавания.
- Написать короткие названия подарков.
- Написать тёплые описания подарков на 1-2 предложения.
- Подготовить визуальный бриф для каждого подарка.
- Проверить, что каждый подарок физически представим как отдельный предмет.
- Согласовать с художником смысл каждого изображения.
- Заполнить `catalog.json`: `id`, `name`, `description`, `imageKey`.
- Проверить, что тексты звучат едино и по-командному.
- Подготовить короткий текст финального поздравления.
- Проверить соответствие `imageKey` реальным файлам.
- Проверить, что в каталоге ровно 50 записей.
- Проверить, что названия не дублируются.
- Проверить, что описания не слишком длинные для мобильного UI.
- Проверить, что каждый подарок подходит для передаривания.
- Подготовить список спорных подарков для финального решения.

## Nice

- Разделить подарки по типам: еда, предметы, офисные мемы, уют, весна.
- Добавить 5-10 запасных подарков на случай замены.
- Добавить более точные промпты для генерации изображений.
- Подготовить короткие подписи для wrapped-состояния, если понадобятся.
- Проверить тексты на одинаковый уровень иронии.
- Проверить, что каталог не выглядит слишком однообразным.
- Добавить лёгкие локальные отсылки к команде.
- Собрать обратную связь от 1-2 коллег до финальной версии.
- Отметить подарки, которые особенно важны для атмосферы.
- Подготовить быстрый changelog замен в каталоге.

## Defer

- Замена слабых подарков после внутренней проверки.
- Сокращение описаний, которые не помещаются в UI.
- Уточнение спорных внутренних шуток после согласования.
- Дополнительная вычитка финального поздравления.
- Финальная сверка каталога с готовыми изображениями.
