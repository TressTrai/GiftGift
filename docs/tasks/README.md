# Задачи по ролям

Короткие production-list файлы по итоговым ролям Pipeline:

- `PL-dev.md` — разработчик.
- `PL-content.md` — куратор подарочного контента.
- `PL-art-ui.md` — художник / дизайнер UI.
- `PL-devops.md` — девопсер.

Внутри каждого файла задачи разделены по приоритетам:

- `Must` — нужно для рабочего запуска.
- `Nice` — полезно, если останется время.
- `Defer` — отложить за пределы первой версии.
