# Interface Contract: Куратор подарочного контента ↔ Художник / Дизайнер UI

## Художник / дизайнер UI ожидает от куратора подарочного контента

- Список 50 подарков с `id`, названием и описанием.
- Короткий визуальный бриф на каждый подарок: что это за предмет и какая деталь делает его узнаваемым.
- Уточнение, какие подарки связаны с конкретной историей и требуют аккуратности.
- Запреты по изображению, если тема может быть неловкой или слишком личной.

Срок: до генерации или отрисовки подарков.

## Куратор подарочного контента ожидает от художника / дизайнера UI

- Изображение для каждого `gift-N`.
- Сохранение смысла подарка из названия и описания.
- Визуальную уникальность подарков.
- Проверку, что предметы не выглядят обидно, гендерно стереотипно или слишком буквально.

Срок: до финальной проверки каталога.
