# Interface Contract: Художник / Дизайнер UI ↔ Разработчик

## Разработчик ожидает от художника / дизайнера UI

- Подарки: `PNG`, `512×512 px`, прозрачный фон, имена `gift-1.png` ... `gift-50.png`.
- Фон сцены: `background.jpg`, `2160×4800 px`.
- Элементы коллективного прогресса: `progress-el-1.png` ... `progress-el-12.png`.
- UI-ассеты: `wrapped-gift.png`, `icon-*.png`, `particle.png`.
- Все ассеты лежат в структуре `public/assets/images/...`.
- Подарки визуально соответствуют `catalog.json` и не меняют смысл предмета.

Срок: до сборки полной игровой петли с реальными ассетами.

## Художник / дизайнер UI ожидает от разработчика

- Финальный список требуемых файлов и путей.
- Стабильные размеры отображения подарков в UI и сцене.
- Информацию, где каждый ассет используется.
- Быструю проверку, что ассеты не обрезаются и читаются на мобильном экране.

Срок: до массовой подготовки ассетов.
