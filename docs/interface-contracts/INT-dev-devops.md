# Interface Contract: Разработчик ↔ Девопсер

## Девопсер ожидает от разработчика

- Рабочий frontend build через `npm run build`.
- Рабочий backend start script.
- `.env.example` для backend.
- Список обязательных переменных окружения.
- Health endpoint `/api/health`.
- Информацию о версии Node.js.
- Отсутствие ручных шагов, не описанных в гайде деплоя.

Срок: до развёртывания на сервере.

## Разработчик ожидает от девопсера

- Публичный HTTPS URL для API.
- Настроенный `VITE_API_URL` на Vercel.
- Запущенный backend process через PM2.
- Рабочий SQLite path по `DB_PATH`.
- Проверенный timezone сервера.
- Доступ к логам backend при сбоях.

Срок: до финальной проверки перед событием.

## Проверка передачи

- `curl https://.../api/health` возвращает успешный ответ.
- Регистрация работает с deployed frontend.
- Логин работает с deployed frontend.
- Игровое состояние загружается после авторизации.
- Мобильный клиент не получает mixed content ошибок.
