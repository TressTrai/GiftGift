## Level 0: Context

**Medium and Platform:**
- Medium: Видеоигра
- Platform: Веб-игра (мобильный фокус)
- Platform constraints: Короткие сессии, простой touch интерфейс, асинхронная игра в течение дня

**Development Constraints:**
- Timeline: 4 дня
- Team: 1 человек
- Budget: ~0₽ (кроме развертывания)
- Tools: Phaser (TS)

**Content Constraints:**
- Rating/audience: Корпоративная аудитория (~20-30 человек из офиса)
- Cultural considerations: 23 февраля = день мужчин (не военная тематика), 8 марта = день женщин
- Accessibility: Простой доступ через браузер на телефоне

**Setting:**
- Setting: Февромарт (объединение 23 февраля и 8 марта)
- Integration level: Cosmetic
- Integration rationale: Праздник как повод для игры, подарки универсальные, можно было бы переделать под другой праздник

**Archetype:**
- Archetype: Party Game (адаптированный для асинхронного формата)
- Inherited commitments:
  - Temporal: Обычно короткие синхронные раунды; адаптируем под асинхронные заходы в течение дня
  - Mechanical: Простые правила, быстрые действия, социальное веселье
  - Cognitive load: Casual (минимальное объяснение, интуитивный интерфейс)
  - Aesthetic affinities: Fellowship, Submission

**LLM Consideration:**
- Relevant: Да, для подготовки контента
- Role: Генерация изображений и описаний подарков ДО релиза
- Implementation: Статический контент (не API интеграция)
- Constraints: Нужно сгенерировать достаточно подарков для разнообразия
