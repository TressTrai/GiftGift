# Система: Получение подарков

**Часть уровня:** 5, см. `Февромарт 5 Обзор.md`.

---

**Что делает игрок:**

1. Видит уведомление о новых подарках.
2. Открывает раздел с нераскрытыми подарками.
3. Выбирает упакованный подарок.
4. Смотрит анимацию раскрытия.
5. Видит содержимое подарка, описание, дарителя, если подарок не анонимный, сообщение и время дарения.
6. После раскрытия подарок попадает в инвентарь.

**Триггеры:**

- другой игрок отправляет подарок;
- игрок выбирает нераскрытый подарок для раскрытия.

**Результат:**

- нераскрытый подарок исчезает из списка новых подарков;
- раскрытый подарок появляется в инвентаре;
- игрок переживает момент сюрприза;
- если подарок подходит к личной тройке целей, связанная система может засчитать прогресс.

**Правила:**

- До раскрытия содержимое подарка скрыто.
- Если подарок анонимный, имя дарителя не показывается.
- Раскрытие обязательно для доступа к содержимому.
- Нераскрытые подарки сохраняются между заходами и не исчезают, если игрок вышел.
- Подарки можно раскрывать в любом порядке.

**Пространственные требования:** отсутствуют; система работает как список нераскрытых подарков и экран раскрытия.

**Парадигма баланса:** Consistency (консистентная), потому что правила скрытия, раскрытия и сохранения подарков фиксированы.

**Активна в сценах:** инвентарь, раскрытие.

- **Активируется:** когда у игрока есть нераскрытые подарки или он открывает раздел новых подарков.
- **Деактивируется:** после раскрытия выбранного подарка или выхода из раздела.
